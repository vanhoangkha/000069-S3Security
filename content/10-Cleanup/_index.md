#### Clean up resources

#### Delete S3 bucket

1. Access to **S3** interface
2. Select **Bucket** related to the lab
3. Select **Delete bucket** (note to **Empty bucket** before deleting)
4. Verify the bucket to be deleted.
5. Select **Delete**

#### Clear CloudFormation Stack

1. Access the interface [CloudFormation](https://console.aws.amazon.com/cloudformation)
2. Select **Stack**
3. Select the stacks related to the lab.
4. Select **Delete**
5. Confirm stack deletion.
