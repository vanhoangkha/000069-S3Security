#### Amazon S3 Bucket Name

1. Access **AWS Management Console**

   - Find **S3**
   - Select **S3**

![Amazon S3 Bucket Name](/images/2.5-BucketName/0001.png?featherlight=false&width=90pc)

2. In the **S3** interface

   - Select **Buckets**
   - Select **sid-security-xxxxxxx** bucket

![Amazon S3 Bucket Name](/images/2.5-BucketName/0002.png?featherlight=false&width=90pc)

3. View details of the **sid-security-xxxxxxx bucket** interface

   - Make a note of the bucket name

![Amazon S3 Bucket Name](/images/2.5-BucketName/0003.png?featherlight=false&width=90pc)
